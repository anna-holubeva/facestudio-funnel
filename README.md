# facestudio-funnel

Статические страницы RU-воронки Face Studio. Хостинг — **GitHub Pages** (сеть Fastly: открывается и в РФ, и в Украине). Домен — **go.facestudio.online**.

## Страницы (live на https://go.facestudio.online)
- `/` — заглушка-корень (`index.html`)
- `/diagnostika/` — лендинг «Бесплатная видео-диагностика лица» (вход в воронку)
- `/rezultaty/` — страница результатов диагностики (разбор 6 тестов + план + подарок)

## Как обновить / задеплоить
См. **[DEPLOY.md](DEPLOY.md)** — там весь процесс (пуш = деплой, где токен, как проверить доступность по странам).

> ⚠️ Репозиторий **публичный** — никаких секретов внутрь. Токены живут в `EN/secrets.local.env`.
